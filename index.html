GTM & Automation Work

Flagship — Growth-Automation Case Study

HireFlow.ai

An n8n pipeline that discovers open GTM / Growth / RevOps roles, enriches each company and hiring manager with Clay and Apollo.io, and drafts personalized outreach with an LLM (Claude) — logging every stage to Airtable and Postgres. A second workflow scores inbound product-usage events into PQLs for PLG teams. A Next.js dashboard visualizes the whole funnel end to end.

Calendar Concierge

n8n Meeting Scheduler

Turns a meeting request — from a web form or an inbound email — into a fully booked Google Calendar event with zero human in the loop. Queries free/busy data to find the first open slot, books the event, and fires a 24-hour, 30-minute, and post-meeting reminder sequence, all from a single n8n execution using Wait nodes.
