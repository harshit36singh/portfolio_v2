# Portfolio v2 — GTM & Automation

My personal portfolio site, focused on GTM, product, and AI-powered automation work (n8n, LLM
orchestration, growth tooling), with Flutter and Spring Boot projects folded in as supporting range.

## Project Structure

```
portfolio_v2/
├── assets/         # Screenshots, resume, and other assets
├── index.html      # Main HTML file
├── style.css       # Styling
└── script.js       # JavaScript functionality
```

## Built With

- HTML5
- CSS3
- JavaScript (vanilla, no framework)

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/harshit36singh/portfolio_v2.git
```

2. Open `index.html` in your browser, or serve it locally:
```bash
cd portfolio_v2
python -m http.server 8000
```
